# Dino Frontier Tycoon

Original dinosaur park management game (inspired by the genre, not by copyrighted assets/code).

## Features
- Build park structures (enclosures, labs, power, hotels, security)
- Hatch herbivore/carnivore dinosaurs
- Guest economy + reputation
- Research tree + unlocks
- Dynamic incidents (storms, sickness, breakouts)
- Save/Load in browser localStorage

## Run
```bash
cd projects/dino-frontier-tycoon
python3 -m http.server 8092
# open http://localhost:8092
```
